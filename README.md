# Dictionary-single-page-application-moringa-module-two
A single page application for a dictionary using wordly and written in HTML5, CSS3, JavaScript.

Kenyan Dictionary SPA

It is a single-page dictionary application. Users can search for a word and instantly see its definitions, pronunciation, audio, examples, and synonyms. All on one page, with no reloads. It uses a public dictionary API and stores favorite words in the browser so they're still there next time you open the app.

**Features**


**Word search**
1. Definitions and parts of speech
2. Pronunciation text
3. Audio pronunciation
4. Examples
5. Synonyms
6. Favorites using localStorage
7. Error handling
8. Responsive design
9. Light/dark theme toggle

**Technologies used**
1. HTML
2. CSS
3. JavaScript
4. Free Dictionary API
5. localStorage
